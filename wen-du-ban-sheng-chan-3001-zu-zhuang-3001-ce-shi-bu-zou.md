# 曾经检查出的错误清单

## 1. 三极管用错的问题

电源电路部分三极管很容易焊错，如果出现程序烧写能发数据，但是上了电池就不能发，电池也确定有充足电量，就一定要检查一遍：Q5Q8器件表面一致为XIL X21，Q3Q9表面一直为ZA C4，D4D5表面一致为XL 3（BAT54C）

器件检查重焊后即可解决，

## 2. 其他易犯的小错误

| 现象 | 解决 |
| :--- | :--- |
| 网口不通 | 少焊了L1，贴片磁珠，贴好后W5500**修复** |
|  | C25焊成了电容，应该改为0欧姆电阻 |
| T0041三极管里将Q3焊成了BAT54C | 去掉重焊为2n3906**修复** |
| 很多板子里Q3Q9被焊为2n3904，也能用， | 暂时没改，是否需要改？ |
| T0043能通能发，无论仿真还是电池都只能发第一次，后面再也不发 | **更换AT8563芯片后修复** |
| T0033能通，但是发出的数据AD=0 | 尚未查明原因 |
| T0043不能工作，按开关就不能发数据，灯亮得很勉强，而且虽然灯亮但没数据到平台。 | 更换电池即可解决。测量ADC小板的J1端子排的1脚与7脚的电压（最两边的两个接线端子），当电池上电开开时的电压必须在3V以上，才能驱动ADC7793芯片。 |
| T0034仿真烧写可以成功，但是总是读到-20 | 铂电阻电路部分R29、R30要去掉；ADC小板上的R7焊的不是2k欧电阻，测出来是3.6k欧，换成2k，修复。 |
